# Keymera

## Who is Keymera

Keymera is a micro-suite for ethical hacking and pentesting.

## Features

- Scan a network (host, ports, SO...)
- Scan users
- Scan tecnologys
- Scan subdomains
- Scan headers
- Scan banners

Read more detailed installations instructions in the [Documentation](https://crakernano.github.io/keymera/)
